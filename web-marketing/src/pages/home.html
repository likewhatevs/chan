quickstart

From install to IDE in two commands

Install the standalone chan CLI, point it at any git repo (or clone chan's own to try it), and the full IDE opens in your browser.

chan serve binds to loopback, prints a URL with a per-launch bearer token, and opens your default browser. Prefer a native window and remote attach? Install Chan Desktop.

$curl -fsSL https://chan.app/install.sh | sh $git clone https://github.com/fiorix/chan $chan serve ./chan

hybrid agent team work

A fleet that works together

Compose a team of agents and watch them run side by side: Claude, Codex, and Gemini in their own terminal sessions, each a different specialist matched to the task. They coordinate with each other through Chan's cs tooling — poke, broadcast, groups — and the built-in MCP server. You conduct from one pane; they collaborate. No in-app chatbot.

scriptable

Drive every pane from the command line

The cs command line reaches into any terminal: write or queue input, broadcast to a group, or pop an interactive survey in another agent's session and read the answer back. Script your whole workflow — or let agents drive each other. The same tooling powers the editor, file browser, graph, and dashboard.

ubiquitous

Server, desktop, local, remote

Run chan wherever you work: a loopback server in your browser or a native desktop window, on your laptop or across the network. Local use needs no account and your files stay ordinary files under the workspace folder. Opt in to the tunnel and the same workspace is reachable from another device — chan serve publishes over an outbound tunnel to a gateway, or Chan Desktop dials a remote chan serve directly over HTTP/2. Local by default, remote by choice.

markdown editor

Where your project's thinking lives

Write your design docs, specs, and tasks in Markdown, with a live outline, cross-file [[wiki-links]], and an inspector that surfaces backlinks and per-file code reports. The documents that drive your work, ready for you and your agents.

Live graph

Navigate the workspace as a graph over directories and files, with layers for Markdown links, tags, and @mentions. Export reproducible chan:// links straight into the editor.

Hybrid search

Find anything with combined BM25 and embedding search across the whole workspace — the same index your agents query through MCP.

Code reports

Per-file and per-directory code analysis (SLOC, COCOMO) in the inspector, so you and your agents can see the shape of the codebase.

Survives reload

Terminal sessions run server-side and tabs are restored, so you can close or reload the window without losing your place.

local-first markdown drives

chan

A desktop and CLI editor for plain markdown folders. Open a drive, write in tabs and panes, link files by path, and keep the source of truth on disk.

Install Read the manual

Plain files

Markdown files stay under the drive folder. Edits from other tools are picked up by the running app.

Path links

Type [[ to pick a file path from the drive. Wiki-links resolve relative paths instead of hiding data in an app database.

Search and graph

Search, backlinks, tags, mentions, and graph views are built from the files in the drive.

Terminal and MCP

Embedded terminals start at the drive root and export CHAN_MCP_* discovery variables for external agent CLIs.

install

Desktop first, CLI when you want a server

The public install path starts with Chan Desktop for local editing. The standalone chan binary remains available for terminal users, remote shells, and explicit chan serve workflows.

macOS DMG Linux AppImage Linux deb

local-first

Local by default, tunnel by choice

Local use needs no account. The HTTP server binds to loopback by default. Tunnel mode is opt-in and publishes the drive through drive.chan.app only when you configure it.

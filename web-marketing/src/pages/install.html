Standalone CLI (headless)

Just the chan CLI and its cs symlink — for a terminal-first chan open, an SSH workflow, or a headless server. The shell installer is the fastest way on macOS or Linux:

$curl -fsSL https://chan.app/install.sh | sh

Prefer a package or a manual download? Pick one below. For install options (a custom prefix, a mirrored metadata source), see the install guide.

After install

$chan open ~/notes

The CLI starts a loopback server, prints the per-launch URL with its bearer token, and serves the editor from the embedded web bundle.

Chan Gateway self-hosting

You don't need this to use Chan. The gateway is the self-hosted tunnel that fronts remote workspaces — install it only if you run your own. The gateway manual covers setup and where it sits in the architecture.

# Search And Graph Basics

Search and graph views are built from the drive contents.

## Search

Search indexes files under the drive root. Use it to find text across the
drive. File/path picking for `[[` links is separate from content search.

## Graph

The graph shows filesystem and markdown relationships derived from the drive.
Filesystem nodes come from files and directories. Markdown edges come from
links, tags, mentions, and contacts where those features are present.

## Freshness

Chan watches the drive for changes and updates the search and graph indexes
after file edits.
